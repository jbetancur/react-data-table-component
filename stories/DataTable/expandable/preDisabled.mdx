# Pre Disabled Rows

To pre-disable rows based on your data:

```js
...
const rowPreDisabled = row => row.disabled;

function MyComponent() {
  return (
    <DataTable
      columns={columns}
      data={data}
			expandableRows
			expandableRowDisabled={rowPreDisabled}
    />
  );
}
```
