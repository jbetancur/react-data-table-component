# Support

## Documentation

The docs cover installation, API, recipes, and examples:
**[reactdatatable.com](https://reactdatatable.com)**

## Bugs and feature requests

Open a [GitHub issue](https://github.com/jbetancur/react-data-table-component/issues) for reproducible bugs or concrete feature proposals. Please include a minimal reproduction.

## Usage questions

Usage questions ("how do I...") are not answered through GitHub issues. Check the [docs](https://reactdatatable.com) and [existing issues](https://github.com/jbetancur/react-data-table-component/issues?q=is%3Aissue) first — most questions are already covered.
